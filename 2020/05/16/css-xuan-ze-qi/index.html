CSS选择器

元素选择器

例：

span {
    background-color: yellow;
}

strong {
    color: rebeccapurple;
}

em {
    color: rebeccapurple;
}

类选择器

例：

.highlight {
    background-color: yellow;
}

类选择器的优势在于灵活，可以通过给元素插入多个类实现更加节省的实现。

注意插入多个类的时候是空格分隔的。

ID选择器

例：

#one {
    background-color: yellow;
}

ID只能被一个元素拥有。

属性选择器

存否和值选择器


虽然属性选择器常常和元素选择器混合使用，但其是可以单独使用的。

例：

li[class] {
    font-size: 200%;
}

[class="a"] {
    background-color: yellow;
}

li[class~="a"] {
    color: red;
}

子字符串匹配选择器


例：

li[class^="a"] {
    font-size: 200%;
}

li[class$="a"] {
    background-color: yellow;
}

li[class*="a"] {
    color: red;
}

大小写不敏感

如果你想在大小写不敏感的情况下，匹配属性值的话，你可以在闭合括号之前，使用i值。这个标记告诉浏览器，要以大小写不敏感的方式匹配ASCII字符。没有了这个标记的话，值会按照文档语言对大小写的处理方式，进行匹配——HTML中是大小写敏感的。

例：

li[class^="a" i] {
    color: red;
}

伪类选择器

伪类是一个选择处于特定状态的元素的选择器，比如它们是这一类型的第一个元素的时候，或者是鼠标指针悬浮在上面的时候。它们表现得会像是你向你的文档的某个部分应用了一个类一样，帮你在你的标记文本中减少多余的类，让你的代码更灵活、更易于维护。

伪类就是开头为冒号的关键字：

:pseudo-class-name

简单伪类

:first-child  /* 第一个元素 */

article :first-child /* article元素内的第一个元素 */

article p:first-child /* article元素内的第一个p元素 */
:only-child  /* 没有任何兄弟元素的元素，等价于下面的形式 */

:first-child:last-child  /* 当然，因为后面这两个用到了更多的选择器，所以权重会大一些 */

:nth-child(1):nth-last-child(1)
:invalid /* 表示任意内容未通过验证的 <input> 或其他 <form> 元素 . */

/* 没有输入正确的值的input元素会被高亮 */
input:invalid {
  background-color: pink;
}

这个伪类对于突出显示用户的字段错误非常有用。

......

用户行为伪类

一些伪类只会在用户以某种方式和文档交互的时候应用。这些用户行为伪类，有时叫做动态伪类，表现得就像是一个类在用户和元素交互的时候加到了元素上一样。案例包括：

:hover  /* 只会在用户将指针挪到元素上的时候才会激活，一般就是链接元素。 */
:focus  /* 只会在用户使用键盘控制，选定元素的时候激活。 */

......

伪元素选择器

伪元素以类似方式表现，不过表现得是像你往标记文本中加入全新的HTML元素一样，而不是向现有的元素上应用类。伪元素开头为双冒号::。

::pseudo-element-name

例：

::first-line /* 选中第一行 */
/* 让第一段的第一行加粗 */
article p:first-child::first-line {
    font-size: 120%;
    font-weight: bold;
}

关于伪元素选择器，还有一对有意思的是::before和::after，两者配合content属性可以做到使用CSS来加入内容：

.box::before {
    content: "This should show before the other content."
}   
<p class="box">Content in the box in my HTML page.</p>

这些伪类的更推荐的用法是插入一个图标，例如下面的示例加入的一个小箭头，作为一个视觉性的提示，而且我们并不希望屏幕阅读器读出它。

还有就是在局部的时候使用：

.box::before {
    content: "";
    display: block;
    width: 100px;
    height: 100px;
    background-color: rebeccapurple;
    border: 1px solid black;
}

全局选择器

例：

* {
    margin: 0;
    padding: 0;
}

以上做法很流行，这样做的好处是会把原来很多代理用户样式表中的元素自带的一些边距、填充值变为0，方便我们之后进行重新的部署。

全局选择器的另一个用途是为了让选择器更加易读。比如下面的例子，我们想要选择所有<article>元素中的第一个，则可以将伪类选择器:first-child设置为其兄弟选择器实现，但这可能导致不好读：

article :first-child {

}

我们可以加上全局选择器使之更加清楚：

article *:first-child { 

}

*运算符（组合选择器）

取交集

即同时符合多个选择器标准的元素 ，直接将多个选择器连接在一起即可：

p.AA   /* class是AA的元素 */

p[class~="a"]   /* 有一个类是a的p元素 */

后代选择器

body article p

子代选择器

article > p

邻接兄弟选择器

p + img /* 紧随在p之后的img标签 */

通用兄弟选择器

p ~ img /* 和p是兄弟关系的img标签，不必与p相邻 */
